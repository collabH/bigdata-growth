# BigData Knowledge Coach Skill

## 概述

这是一个专为大数据学习者打造的智能助手Skill，基于个人大数据知识库构建，帮助你高效复习知识、准备面试和系统学习大数据技术栈。

## 功能特性

### 🎯 核心功能

1. **知识点复习** - 快速回顾和理解大数据技术要点
2. **面试准备** - 提供分层次的面试题库和模拟面试
3. **学习路径规划** - 制定个性化的学习计划
4. **知识点对比** - 清晰对比不同技术的优劣和适用场景
5. **问题诊断与调优** - 解决实际遇到的技术问题

### 📚 覆盖的技术栈

- **计算引擎**: Flink、Spark
- **数据存储**: HDFS、HBase、RocksDB、Zookeeper
- **数据湖**: Hudi、Iceberg、Paimon
- **OLAP引擎**: Hive、Kudu、Impala、ClickHouse、Druid、Kylin
- **消息队列**: Kafka、Pulsar
- **数据采集**: Canal、Debezium、Flume
- **任务调度**: Azkaban、DolphinScheduler
- **数据仓库**: 建模理论、实时/离线数仓、数据中台

## 交互原则

### 1. 主动引用文档

每次回答都会引用相关的文档路径，方便你深入学习：
```markdown
**相关文档：**
- [文档标题](相对路径)
```

### 2. 分层回答

- **第一层**：直接回答问题（简洁明了）
- **第二层**：深入原理解释（按需展开）
- **第三层**：源码级别分析（针对高级问题）
- **第四层**：生产实践建议（结合实际场景）

### 3. 引导式学习

不会一次性灌输所有知识，而是：
- 先给框架，再深入细节
- 提出问题，引导思考
- 根据你的反馈调整深度

### 4. 实战导向

- 强调生产环境的最佳实践
- 分享踩坑经验
- 提供可操作的调优建议

## 学习建议

### 高效使用技巧

1. **明确目标**
   - 告诉助手你的当前水平和学习目标
   - 例如："我是Java开发，想转行做Flink开发"

2. **循序渐进**
   - 按照推荐的学习路径逐步学习
   - 不要跳跃式学习

3. **理论与实践结合**
   - 理论学习后立即动手实践
   - 记录遇到的问题和解决方案

4. **定期复习**
   - 使用间隔重复法复习
   - 制作自己的知识卡片

5. **主动提问**
   - 不懂就问，不要积累疑问
   - 追问"为什么"，深入理解原理

## 常见问题

### Q: 这个Skill适合什么水平的人使用？

A: 适合所有水平的学习者：
- **初级**：提供基础概念和学习路径
- **中级**：深入原理和实践技巧
- **高级**：源码分析和架构设计

### Q: 如何最大化利用这个Skill？

A: 
1. 把它当作私人导师，随时提问
2. 按照推荐的学习路径系统学习
3. 定期用它进行知识复习
4. 面试前用它模拟面试
5. 遇到问题时用它诊断和解决

### Q: 知识库会更新吗？

A: 是的，随着你的学习和实践，可以持续向知识库添加新内容，Skill会自动利用这些新知识。

### Q: 可以自定义学习内容吗？

A: 当然可以！你可以：
- 指定想学习的特定技术
- 要求调整学习难度
- 提出特定的业务场景
- 请求针对性的面试题
